# GarageParkAssist
For Arduino Nano - NeoPixel garage assistant

Uses Adafruit NeoPixel example code to give colored/animated car distance feedback:

|95"|<------BLUE------>|44"|<-------GREEN------->|32"|<------RED------->|18"|<-------RED WITH WHITE X------->|WALL|

nearLimit = 82cm = 32"  
farLimit = 243cm = 95"  
midLimit = 112cm = 44"  
dangerLimit = 45cm = 17.7"
